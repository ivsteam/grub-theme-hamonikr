# hamonikr-plymouth-theme

grub theme for HamoniKR 

# Developer

Kevin Kim <root@hamonikr.org>

# Installation

```
$ wget https://github.com/ivsteam/grub-theme-hamonikr/archive/1.0.tar.gz
$ tar -xvzf 1.0.tar.gz
$ cd grub-theme-hamonikr
$ sudo ./install
```
